顯而易見的好處，幫你 收納 Template 中的邏輯，給予邏輯更為明確的命名。

計算屬性(Computed) 另一個好處「其相依 data 一變，computed 也會隨之更新」，反過來講，相依 data 不變就不會重複計算(反正結果也一樣)，computed直接吐回結果。

輸入姓氏(v-model)

{{ '{' + '{firstName}' + '}' }} = {{ firstName }}

{{ '{' + '{lastName}' + '}' }} = {{ lastName }}
