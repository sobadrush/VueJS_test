test-008 Component - filter(過濾器)

Filter 有分【局部註冊】【全域註冊】

Filter 是有呼叫順序的

Filter 可以傳參數

{{ '{' + '{ gameName }' + '}' }} ... 使用 globalFirst2Chars 過濾 >>> {{ gameName | globalFirst2Chars}}

{{ '{' + '{ gameName }' + '}' }} ... 使用 globalFirst1Chars 過濾 >>> {{ gameName | globalFirst1Chars}}

{{ '{' + '{ gameName }' + '}' }} ... 使用 globalPrompt 過濾 >>> {{ gameName | globalPrompt( 'Fuck' , 'Mother' )}}

{{ '{' + '{ gamePrice }' + '}' }} ... 使用 priceFormatJpyBefore、priceFormatJpyAfter >>> {{ gamePrice | priceFormatJpyBefore | priceFormatJpyAfter }}
